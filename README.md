# 🏢 InnovaNexus

## 📖 About The Project
InnovaNexus is the management portal and administrative counterpart to **[InnovaStay](https://github.com/Muhammad-Dawood-Zulfiqar/InnovaStay-Android-Hotel-Booking-App)**. This platform is designed specifically for hotel owners and managers, giving them full control over their property listings and the ability to process incoming booking requests in real-time.

## 💻 Tech Stack
* **Frontend:** XML
* **Backend:** Java
* **Database:** Firebase

### 🏘️ Property Management
<img src="https://github.com/user-attachments/assets/53b3ea61-7cae-44e0-b5f6-f8d42d4dd2e7" alt="InnovaNexus Property Management" width="250" align="right" />
Owners have complete CRUD (Create, Read, Update, Delete) capabilities over their hotel listings. They can add new properties, update descriptions, and manage pricing to keep their InnovaStay listings accurate.

<br clear="right"/>
<br>

### 📥 Centralized Booking Dashboard
<img src="https://github.com/user-attachments/assets/b173c1c9-0842-4958-9dc4-e86bc229ac3f" alt="InnovaStay See Bookings" width="250" align="right" />
All incoming reservations from InnovaStay users are routed directly to the owner's dashboard, providing a clear overview of pending, upcoming, and past bookings.

<br clear="right"/>
<br>

### ✅ Booking Action Hub
<img src="https://github.com/user-attachments/assets/7c8a3c9b-6b7b-4ed8-89ff-f024a036ff6f" alt="InnovaNexus Manage Bookings" width="250" align="right" />
Owners maintain full control over their availability by manually reviewing incoming requests and choosing to **Approve** or **Reject** them. These status updates are immediately reflected on the user's InnovaStay app.
